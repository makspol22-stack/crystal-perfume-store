# Crystal Perfume Store
Premium селективная парфюмерия с энергией натуральных камней.

## О проекте
Сайт Crystal Perfume — это современный интернет-магазин, объединяющий искусство парфюмерии и литотерапию. 

### Основные возможности:
- **Динамический каталог**: Более 111+ ароматов с "аурой" и подробным описанием.
- **Интерактивный подбор**: Выбор аромата по энергетике камня.
- **Корзина и заказ**: Оформление заказа напрямую в WhatsApp.
- **Премиальный дизайн**: Светлая тема, микро-анимации и стеклянный размыв (glassmorphism).

## Технологии
- HTML5 / CSS3 (Modern standards)
- Vanilla JavaScript
- Font Awesome 6
- GitHub MCP Integration
